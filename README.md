# Retail Sales Analysis - Accessories Store

Repositori ini berisi proyek analisis data sederhana untuk memantau performa penjualan di bisnis aksesori retail. Proyek ini bertujuan untuk mengubah data mentah menjadi wawasan bisnis yang dapat ditindaklanjuti.

## Tools dan Library
* Python: Bahasa pemrograman utama.
* Pandas: Digunakan untuk manipulasi dan pembersihan data.
* Matplotlib: Digunakan untuk visualisasi data dalam bentuk grafik.

## Analisis yang Dilakukan
1. Total Revenue Calculation: Menghitung total pendapatan keseluruhan.
2. Best-Selling Products: Mengidentifikasi produk mana yang paling banyak terjual.
3. Revenue per Product: Menganalisis kontribusi pendapatan dari tiap kategori produk.
4. Data Visualization: Menampilkan tren penjualan menggunakan grafik batang (bar chart) agar lebih mudah dipahami.

## Business Insight
Berdasarkan hasil analisis, produk dengan pendapatan tertinggi merupakan kontributor utama terhadap profit perusahaan. Insight ini disarankan untuk digunakan dalam:
* Manajemen Stok: Memastikan stok produk terlaris selalu tersedia.
* Strategi Promosi: Memberikan promo bundel pada produk yang kurang laku dengan produk best-seller.
